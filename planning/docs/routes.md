# Routes & Views

## /

**Purpose:** Home page showing a list of articles.  
**URL params (query):** `topic`, `sort_by`, `order`, `page`  
**Primary components on page:** Header, TopicFilter, SortControls, ArticlesList, Pagination

## /articles/:article_id

**Purpose:** Single article page.  
**URL params (query):** none  
**Primary components on page:** Header, Article, Votes, CommentList, NewComment
